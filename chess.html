Chess Playing Robot

Time: Spring 2023
Role: Team lead, Mechanical-electrical engineer
Tools: Hand tools, SolidWorks, 3D Printing

SUMMARY

This is a personal project built by a team of two, including myself. The robot is a single arm 3D printed with a camera overhead looking down at the chess board. It takes a picture each time a move is complete, then decides how to move using chess engine Stockfish. While there are many versions of this robot online, this one was chosen specifically to have an independent robot that does not need any sensors on the board itself, and thus can be more versatile.
